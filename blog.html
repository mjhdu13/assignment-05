Question-1: What are the different ways to select an element in the DOM?

We can select element in the DOM in various ways. Some of the are: getElementById(), getElementsByClassName(), getElementsByClassName(), querySelector(), querySelectorAll() etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

We can manipulate the HTML within an element by selecting innerHTML. On the other hand, we can manipulate the visible text inside an element by selecting innerText. We use textContent when we need to work with the text content only, if it is visible on not.
